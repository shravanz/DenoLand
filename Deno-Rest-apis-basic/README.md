A Simple CRUD REST APIs service with hard-coded values in the memory

1. git clone 
2. run the Deno server 
```
 deno run --allow-net server.ts
```
3. Routes
```
GET      /api/v1/products
GET      /api/v1/product/:id
POST     /api/v1/product
PUT      /api/v1/product/:id
DELETE   /api/v1/product/:id
```
